# Module 10 Assignment

This repository contains my work for Module 10. The assignment involved cleaning and transforming a messy real-world customer dataset from UrbanStyle into a structured format ready for analysis.

## Files Included

- module10-assignment.py

## What I Practiced

In this assignment, I practiced handling missing data with .fillna() and .dropna(), converting data types with .astype() and pd.to_datetime(), cleaning text data using the .str accessor, creating derived features with .apply(), removing duplicates, and generating business insights using groupby aggregations and correlation analysis.
